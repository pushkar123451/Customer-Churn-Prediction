# 📊 Customer Churn Prediction

## 🚀 Project Overview
This project aims to analyze customer churn in a telecom company and build a machine learning model to predict customers who are likely to leave.

---

## 📂 Dataset
- Telco Customer Churn Dataset
- 7000+ records, 20 features

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

---

## 🔍 Exploratory Data Analysis (EDA)
- Analyzed churn distribution
- Studied impact of tenure, monthly charges, and contract type
- Identified key churn patterns

---

## 🤖 Model Building
- Used Random Forest Classifier
- Applied train-test split (80-20)
- Performed feature encoding and preprocessing

---

## 📈 Results
- Accuracy: 79%
- Recall: 88% (focused on identifying at-risk customers)

---

## 💡 Key Insights
- Customers with low tenure are more likely to churn
- High monthly charges increase churn probability
- Month-to-month contracts have the highest churn rate

---

## 📊 Business Recommendations
- Improve onboarding for new customers
- Offer discounts for high-paying customers
- Promote long-term contracts

---

## 🛠️ Future Improvements
- Hyperparameter tuning
- Use of advanced models (XGBoost)
- Deployment using Streamlit
